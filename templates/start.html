About the United States Wind Turbines Data


The data used in this web app is taken from here and can be found on the CORGIS Dataset Project website.


One statistic in this dataset is the turbine capacity. This refers to the amount of electricity that a single turbine can produce, and is measured in kilowatts (KW). So, if a turbine has a capacity of 65, that means that it can produce up to 65 kilowatts of power. However, there is also a statistic called the project capacity. This number refers to how much electricity every turbine in a project (a group of turbines built at the same time) can generate all together, measured in megawatts. So, if there was a project with 10 turbines that could each produce 65 kilowatts of power, the project capacity for each turbine would be 0.65, indicating that the entire project at peak performance could generate 0.65 megawatts of electrical power.
